## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/AliiAnr/Ali-Personal-Portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   ```
   http://localhost:5173
   ```

### Build for Production

```bash
npm run build
# or
yarn build
```

### Preview Production Build

```bash
npm run preview
# or
yarn preview
```

## ğŸ¨ Customization

### Personal Information
Update your details in `src/data/user.js`:
```javascript
export const user = {
  name: "Your Name",
  title: "Your Title",
  location: "Your Location",
  // ...
};
```

### Projects
Add your projects in `src/data/projects.js`:
```javascript
export const projects = [
  {
    id: 1,
    title: "Project Name",
    description: "Project description...",
    // ...
  }
];
```

### Theme Colors
Customize colors in `tailwind.config.js` and CSS variables in `src/index.css`.
